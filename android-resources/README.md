# Android Resources

## App Icon
Place your app icon files in:
- android/app/src/main/res/mipmap-hdpi/ic_launcher.png (72x72)
- android/app/src/main/res/mipmap-mdpi/ic_launcher.png (48x48)
- android/app/src/main/res/mipmap-xhdpi/ic_launcher.png (96x96)
- android/app/src/main/res/mipmap-xxhdpi/ic_launcher.png (144x144)
- android/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png (192x192)

## Splash Screen
The splash screen uses the backgroundColor from capacitor.config.ts (#1A1A2E)
with the app name displayed.

## To generate icons from the logo:
Use Android Studio > Image Asset Studio
Or https://icon.kitchen/ (free online tool)
